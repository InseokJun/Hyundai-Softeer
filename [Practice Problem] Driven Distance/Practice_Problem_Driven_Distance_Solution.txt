[문제 접근법 및 풀이 방법]


--- 고려 사항 1 ---

대소 관계에 대한 조건문 분기를 할 줄 알아야한다. 


[변수 의미]

변수의 의미는 최대한 변수 이름에 반영하도록 노력하였다. 

- distance_list => Input Data로 a와 b를 담고 있는 List를 의미한다. 
- a => Input Format에 맞는 Input Data로 공백을 기준으로 앞에 위치한 정수를 의미한다. 
- b => Input Format에 맞는 Input Data로 공백을 기준으로 뒤에 위치한 정수를 의미한다.



[풀이 방법 설명]

1. 문제에서 주어진 입력 형태에 맞도록 Input Data를 받아온다. 

2. 받아온 Input Data를 문제에서 요구한 형태로 대소 관계 비교 연산을 수행한다. 

3. 대소 관계에 따른 지정된 출력을 나타내도록 한다. 



[알아두면 좋은 문법] 

- If .. elif .. else 문법에 대해 알고 있으면 좋다. 



[얻은 교훈]

- 각 Case에 대한 조건 분기를 효율적으로 나타낼 수 있어야한다. 
- Input Data를 받아오는데 다양한 방법이 존재하며 이는 시간 측면에서 영향을 준다.
