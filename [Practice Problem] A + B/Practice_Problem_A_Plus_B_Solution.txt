[문제 접근법 및 풀이 방법]


--- 고려 사항 1 ---

덧셈 연산 수행을 할 줄 알아야한다. 

--- 고려 사항 2 ---

Python 다양한 Print문 작성 방법을 알아야한다. 



[변수 의미]

변수의 의미는 최대한 변수 이름에 반영하도록 노력하였다. 

- case_num => 덧셈 연산을 수행할 횟수를 의미하며 이는 단순한 정수의 형태로 존재한다. 
- a => Input Format에 맞는 Input Data로 공백을 기준으로 앞에 위치한 정수로 피연산자를 의미한다. 
- b => Input Format에 맞는 Input Data로 공백을 기준으로 뒤에 위치한 정수로 피연산자를 의미한다.



[풀이 방법 설명]

1. 문제에서 주어진 입력 형태에 맞도록 Input Data를 받아온다. 

2. 받아온 Input Data를 문제에서 요구한 형태로 덧셈 연산을 수행한다. 

3. 지정된 출력 형식에 맞게 출력하도록 한다. 



[알아두면 좋은 문법] 

- print문에 f".."를 작성하는 방법을 알고 있으면 좋다. (String Formatting)



[얻은 교훈]

- 출력문을 작성할 때 다양한 String Formatting을 알고 있으면 효율적인 출력 코드 작성이 가능하다.
